## 📌 About

This repository contains implementations of fundamental algorithms categorized by their approach:

- 🔁 **Backtracking** → Exhaustive search with constraints  
- ⚡ **Divide and Conquer** → Breaking problems into smaller subproblems  
- 🧠 **Dynamic Programming** → Optimization using memoization/tabulation  
- 🌐 **Graphs** → Graph traversal & coloring techniques  
- 💰 **Greedy** → Locally optimal choices for global optimum  

Each `.cpp` file contains a clean and focused solution to a classic problem.
